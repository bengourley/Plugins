Draggy Switch is: Off

Description:

A draggable UI switch element. Works in all browsers including Mobile Safari. Takes optional callbacks to fire when switching on/off, can be set to run once. Listens for a custom event to fire so it can be reset asynchronously.

Usage:

$(".drag-range").dragToEnvoke({
	onFunc : function() {
		$("h1 span").text("On");
	},
	offFunc : function() {
		$("h1 span").text("Off");
	}
});

See source for all options.
